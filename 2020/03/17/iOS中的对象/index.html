Class 意为一个不透明的类型（指针） 代表一个Objective-C的类

id 意为类的实例的指针

SEL 意为一个不透明的类型 代表一个方法选择器

IMP 意为方法实现的函数指针

如果有swift的代码。则可变的IMP是不透明指针，不可变则不是。

OBJC_TYPES_DEFINED` 宏定义是因为在另一处
